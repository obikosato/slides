## Editing

### Requirements

- [Marp CLI](https://github.com/marp-team/marp-cli)

### Commands

```sh
# Serverモードでプレビュー
make browse

# HTMLに変換
make html

# PDFに変換
make pdf

# distディレクトリをクリア
make clean
```
